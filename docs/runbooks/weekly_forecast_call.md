# Weekly CCO Forecast Call Runbook

## Purpose

Make the weekly forecast call decision-oriented. The call should not be a narration of dashboards; it should expose commit risk, assign owners, and close follow-ups.

## Inputs due Friday EOD

- Regional forecast submissions
- Commit / best-case / upside movements
- New pipeline created this week
- Pipeline slipped this week
- Close-date pushes
- Stage regressions
- Large deal notes
- Manager inspection asks

## Standard agenda

1. Submission completeness check
2. Segment-level forecast coverage
3. Commit movement and deltas vs prior week
4. High-risk segment inspection
5. Large-deal pull-in / slip review
6. Pipeline hygiene exceptions
7. Action owner assignment
8. Decision log confirmation

## Output artifacts

- Forecast call pre-read
- Forecast decision log
- Updated action tracker
- Risk memo for CCO staff
