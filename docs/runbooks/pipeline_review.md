# Pipeline Review Runbook

## Purpose

Ensure future-quarter pipeline is sufficient, high quality, and aligned to the annual plan.

## Inputs

- Pipeline required by segment and quarter
- New pipeline created
- Source mix
- MQL / SQL / SAO conversion rates
- BDR meeting and opportunity creation metrics
- Pipeline hygiene score
- Territory whitespace and focus account list

## Decision questions

- Is pipeline coverage above the required threshold?
- Is pipeline quality improving or deteriorating?
- Which segments need demand-gen or BDR intervention?
- Which territories require focus campaigns?
- Which pipeline sources are underperforming?

## Output artifacts

- Segment pipeline gap list
- Demand-gen action plan
- BDR capacity / activity actions
- Territory focus list
