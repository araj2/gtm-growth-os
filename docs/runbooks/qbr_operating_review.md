# QBR / GTM Operating Review Runbook

## Purpose

Connect quarterly performance, pipeline quality, capacity, territories, comp, and operating standards into one leadership review.

## Required sections

1. ARR bridge and bookings performance
2. Pipeline generation and conversion
3. Segment productivity
4. AE capacity, ramp, and quota coverage
5. Territory balance and whitespace
6. Demand-gen and BDR contribution
7. Forecast accuracy and slippage
8. Process divergence / standards adoption
9. Decisions required for the next quarter

## Output artifacts

- QBR executive readout
- Next-quarter operating plan
- Standards / process updates
- Action tracker and decision log
